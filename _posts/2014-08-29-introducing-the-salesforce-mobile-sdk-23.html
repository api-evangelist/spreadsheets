The Mobile SDK 2.3 is out!This release includes significant updates for developers creating hybrid applications.Salesforce specific functionality for hybrid apps are now available as independent Cordova plugins.Developers can consume Salesforce OAuth2 Identity, SmartStore Secure Offline Database, and Push plugins in the very latest versions of Cordova independent of Mobile SDK releases.
