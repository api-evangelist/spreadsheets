Today, partners can use Trialforce to provision trials which is an easy and seamless way to turn prospects into paying customers.The new Signup Request API  takes this functionality to the next level. It exposes a new SignupRequest object via the API and by simply adding records to this object, trials can be created.Since this is a standard object, it provides you with the full power of the Force.com platform.
