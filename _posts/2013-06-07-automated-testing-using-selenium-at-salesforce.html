Testing is an important part of releasing quality software.Automated software testing can execute those routine, monotonous cases for you, freeing up your testing resources to concentrate on more interesting cases in your application.As part of a well-constructed automated test pyramid, you want to have some level of UI-based automated tests.For web applications, that means automating the browser.There are a number of tools you can use to automate testing of the browser.
