Were excited to announce new features to help you discover and customize kimono APIs extracting data from across the web: API search: Search our fast growing database of over 50,000 public kimono APIs to find the data you want.Search for real estate, jobs or bitcoin APIs to get started.Existing API panel: Load extraction rules from other APIs built from the webpage you are kimonifying.When you launch kimono, you will see a panel showing existing API templates.Click one to load it in your browser.
